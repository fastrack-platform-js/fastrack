# Security

## Reporting vulnerabilities

Please report security vulnerabilities privately. Do not open a public issue.

- Prefer [GitHub Security Advisories](https://github.com/fastrack-platform-js/fastrack/security/advisories/new) for this repository, or
- Contact the maintainers via a private channel if needed.

We will acknowledge and respond as soon as possible.

See also [docs/security.md](docs/security.md) for platform security (JWT in-service, no PII in logs, readiness).
